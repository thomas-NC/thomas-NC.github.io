Editeur de Code

Cette page est l'équivalent d'un JsFiddle, mais seulement pour du code Javascript. En gros le principe est que l'utilisateur peut écrire son propre code à partir de la page en utilisant les boutons de gauche ou la zone de texte à droite. le code est ensuite lancé a partir de la page et affiche un résultat en bas à droite. Ce projet servait d'introduction à Javascript.

Projets Jeu Vidéo

Baby Kraken

Baby Kraken est le jeu qui m'a introduit à Unity. C'est un petit jeu 2D rageant dont le but est d'utiliser la souris pour cliquer sur un cadenas sans se faire toucher par les tentacules. Je n'avais jamais touché à Unity avant ca, et n'avais jamais utilié C#, qui est le langage de base pour les scripts. La principale difficulté était d'animer un tentacule de façon à lui donner des mouvements réalistes. Le jeu en soi a été terminé en deux semaines de travail à peu près, et devrait sortir d'ici quelques semaines sur un page web au nom de "Baby Kraken".

Cosmo Quest

Cosmo Quest est un jeu issu de la game jam à laquelle j'ai participé en fevrier 2018. La contrainte était "confrontation entre la 2D et la 3D", la mécanique principale du jeu est donc d'appuyer sur un bouton qui fait passer le jeu d'un mode 2D à un mode 3D. Mon équipe et moi n'avons pas pu finir le jeu en 26h, cependant il a suffit a nous faire gagner cette game jam. A l'heure actuelle le jeu est encore en développement et sortira très certainement au cours de l'année.
