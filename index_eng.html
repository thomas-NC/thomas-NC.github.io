Video Game Projects

Baby Kraken

This is the video game I worked as our biggest project of the year. It's a 2d game made to be played on a computer, it's a rage inducing game that can be quite unfair, but is always lots of fun. Cosmo Quest is a game that was made for a Game Jam, the requirements were "confrontation between 2d and 3d world", this is a platformer developped in Unity 3d. The main game mechanic is that you can press a key to switch between dimensions. The game is still under developpment but it will definitly see the light of day sometime.
